# Sign Language Character Recognition

Trained a Convolutional Neural Network (CNN) to recognize and translate sign language gestures into text based on the Sign Language MNIST dataset.

## Quick Info
- Task: Image Classification / Deep Learning
- Dataset: Sign Language MNIST
- Notebook: `sign_language_character_recognition.ipynb`
- Exports: `exports/` (HTML)

## How To Run
- Open the notebook and run all cells from top to bottom.
- If a dataset file is required, place it in this folder.
